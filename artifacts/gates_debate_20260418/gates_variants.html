Gates · visual language candidates

0 · Original (yours)

10pt white trunk · 1pt status stripe above · squiggle for gate · vertical bar for broken

Three visual vocabularies (line / squiggle / bar). Squiggle dies at small zoom; 1pt stripe fails on projector.

1 · Geometric split

Shape encodes structure (gate or not) · Color encodes runtime state · one vocabulary

Diamond glyph = gate (like a circuit symbol). Color carries runtime state only. One alphabet, two channels.

2 · Notch-only

Gate = notch cut into the trunk · no stripe at all · pure monochrome + single accent

Most austere. Gate is a symbol *in* the flow, not an annotation above it. Red used only for failure — rare, attention-grabbing.

3 · Fill + stripe

Trunk fills in as the run progresses · stripe above shows gate status · preserves your original feel

Trunk opacity encodes progress (0/50/100%). Stripe above still encodes gate state. Amber dashed = pending gate. Reads as a timeline.

Shared vocabulary (all variants)

trunk · flow passed pending failed / hard-stop ◇ gate present ✕ hard stop
